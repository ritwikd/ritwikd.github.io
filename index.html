Projects

securewallet

SecureWallet is a cool little password manager I wrote as a side project. It uses 128-bit AES encryption. It's written in pure Python, but I've only tested on Linux.

rpn-calc

rpn-calc is a neat little command-line reverse Polish notation calculator that I wrote one day because I thought it was interesting idea.
